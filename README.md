# Medi-Rdv

**Medi-rdv** is a web application for making online medical appointments powered by **[Node-Red](https://nodered.org)**.

![logo](preview/rdv-logo.png)

## Tools

To make this applicaiton, we used many tools amount which:

- HTML
- CSS
- Bootstrap
- JavaScript
- MySQL
- AWS SES
